01.About The Study

In an era where technology is rapidly evolving, the gap between theoretical knowledge and practical application in vocational schools like Gen. Flaviano Senior High School is widening.


This research, specifically targeting CSS Grade 12 Students (Batch 2025-2026), investigates the impact of a dedicated web platform named CTRL (Computer Troubleshooting & Resource Learning).


The study aims to determine if integrating a structured troubleshooting website into the curriculum can significantly enhance the students' technical proficiency and self-efficacy in handling real-world hardware and software issues.

02.Tutorial

Computer Components

Computer components are the essential parts that work together to make a computer function properly. The main hardware components include the Central Processing Unit (CPU), which acts as the brain of the computer; the motherboard, which connects all parts together; Random Access Memory (RAM), which temporarily stores data for quick access; storage devices such as Hard Disk Drives (HDD) or Solid-State Drives (SSD) that store files and programs; and the Power Supply Unit (PSU), which provides electricity to the system. Input and output devices like the keyboard, mouse, and monitor also play an important role in allowing users to interact with the computer.

PC Troubleshooting: Basic

Basic PC troubleshooting involves identifying and solving common computer problems. These problems may include the computer not turning on, slow performance, system freezing, or no display on the monitor. Simple troubleshooting steps include checking all cable connections, restarting the computer, making sure hardware components are properly connected, scanning for viruses, and updating drivers or software.
